# ROS2 vs. ROS1  
Different:  
- ROS2: No need of `rosmaster` anymore. Using `DDS` communication mechanism.  
- ROS2: More OOP-style programming.  
  
Similar:  
- Similar framework and communication concepts: topic, service, action etc.
